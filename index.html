Changing width and height of Font in Realtime
NOTE - This web application can change the font size of a text by using Left Wrist and Right Wrist. Closer the Left Wrist and Right Wrist to each other smaller the font size, farther the Left Wrist and Right Wrist to each other bigger the font size.
